What can a Data-Whiz do?

Everything dealing with Data : Getting, Analyzing and Deploying.

  • GETTING DATA
  • ANALYZING
  • DEPLOYING

Web Scraping

- Scourge the web for openly available Data and APIs.
- Create own dataset by scraping data from Websites, Forums and Social Networks.

Quality Control

- The success of a project depends a lot on the quality of the dataset. 
- Do rigorous control to check if the data is correct and complete. 

Data Cleaning

- Combine multiple sources to verify and complete the data
- Intelligent imputation of missing data based on context (spatial, time, or other features)

Machine Learning

- Use data and create new features to predict  with high accuracy and reliability.
- Develop Explainable Models which can pass smell test by domain experts.

Deep Learning

- Use state of the art Neural Network Architectures for Computer Vision Tasks.
- Language Modelling using Recurrent Neural Networks.

Business Analytics

- Get business insights into data through market segmentation and CRM data.
- Build Recommendation Engines for media and e-commerce platforms

Creating Dashboards

- Present pertinent data via Maps and Charts for ease of understanding.
- Make data interactive and easy to explore.
- Simplify data by defining new metrics / KPIs

Deploying Models

- Deploy Machine Lerning/Deep Learning models using REST APIs.
- Create Applications with the models.
- Design the Application Architecture.

Big Data

- Evaluating the need for Big Data.
- Choosing the right tools and architecure to scale up the application.
- Machine Learning on Distributed Databases.
