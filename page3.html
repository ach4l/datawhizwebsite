What will you learn?

Everything dealing with Data : Getting, Analyzing and Deploying.

  • GETTING DATA
  • ANALYZING
  • DEPLOYING

Web Scraping

- Create own dataset by scraping data from Websites, Forums and Social Networks.
- Technology : BeautifulSoup, 'Hacking' websites to get data from their backend.

Data Exploration

- Do rigorous control to check if the data is correct and complete. 
- Technology : Dataiku, Python Visualisation Libraries (matplotlib / Seaborn)

Data Manipulation

- Manipulate data to get into the right format for Data Analysis
- Technology : Pandas, Numpy, Dataiku

Machine Learning

- Different types of Machine Learning (Supervised and Unsupervised)
- Different Tasks (Regression, Classification and Recommendation)

Algorithms

- Various Algorithms like Linear Regression, Logistic Regression, Random Forests, Boosting Algorithms, Neural Networks
- Technology - Dataiku, Tensorflow (Keras)

Different Data Types

- Learn how to handle different kinds of dataset
- Tasks with Natural Language Processing (NLP), Image Data, and Time Series forecast
- Technology - Natural Language Toolkit (nltk)

Creating Dashboards

- Present pertinent data via Maps and Charts for ease of understanding.
- Make data interactive and easy to explore.
- Technology - Google Data Studio, Dataiku

Deploying Models

- Deploy Machine Lerning/Deep Learning models using REST APIs.
- Create Applications architecture.
- Technology - Flask, heroku

Creating Applications

- Quickly creating frontend for applications
-  Progressive Web Application (PWA) that can also be installed as a phone application
- Technology - Bootstrap, Html

FAQ - BootCamp

  How is "Data-Whiz" different from Data Scientist?

Someone who knows only Machine Learning Algorithms can call himself/herself a Data Scientist. A Data-Whiz, however, can handle end-to-end Data projects; from Data Scraping to Creating Data Applications. Machine Learning is a part of his skills, but he can do much more!

 Do you have any evening courses?

Short answer - Yes, from August onwards. However, we strongly encourage learning through Intensive Bootcamp as it is the fastest and most effective way to learn. 

 How does the 'guaranteed' internship work?

Internship is guaranteed for everyone who successfully completes the BootCamp. However, it is free for those who finish in top 20% of the bootcamp. For everyone else, you can choose to do an internship by paying ₹ 1,000 for a two-week internship. The internship can be availed at any time in the next 6 months.

You will be given innovative, real-life projects during the internship which will help you practice and cement your skills immediately. For those selected to work on company projects, a stipend will be given based on project deliverables.

 Would you teach Programming as well?

.We will follow a two-pronged approach : 

- Use fantastic tools like Dataiku which let you do almost all the tasks without needing to write any code. You should be able to successfully complete the course without any coding if you want to.

- For those who know little bit programming, and for those who want to learn, we will provide template codes which you can fill in to do the same tasks as with the tools.

Which approach you choose depends completely on you.

 What will be the class timings?

Theory Classes - 9:30 AM - 12:30 PM
Practical Classes - 2 PM - 6 PM
